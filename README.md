# Student Placement Prediction Engine 🎓💼

An end-to-end Machine Learning classification project that analyzes student performance records and predicts campus placement possibilities using predictive modeling.

## 📊 Performance Metrics Tracked
* **CGPA:** Cumulative academic score.
* **Aptitude & Communication:** Technical and soft-skill evaluation ratings.
* **Projects & Internships:** Practical implementation and corporate exposures.

## 🛠️ Tech Stack & Tools Used
* **Python** (Core Language)
* **Pandas** (Data Manipulation & Encoding)
* **Scikit-Learn** (Logistic Regression, Model Selection & Evaluation Metrics)

## 📈 Model Evaluation Summary
The model tracks performance using advanced diagnostic tools:
* **Accuracy Score:** Overall classification success percentage.
* **Confusion Matrix:** True Positives, True Negatives, and Error breakdowns.
* **Classification Report:** Detailed class-wise Precision, Recall, and F1-Score percentages.
